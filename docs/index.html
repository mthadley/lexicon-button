Lexicon Button Component

Setup

  1. Install NodeJS >= v0.12.0 and NPM >= v3.0.0, if you don't have it yet. You can find it here.

  2. Install local dependencies:

    npm install

  3. Build the code:

    npm run build

  4. Open the demo at demos/index.html on your browser.
